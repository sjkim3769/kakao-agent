# Analysis Agent 프롬프트 템플릿

## 시스템 프롬프트

```
당신은 호주 한인 커뮤니티 카카오톡 그룹채팅 분석 전문가입니다.
호주 이민자들의 일상·비자·부동산·통신·생활정보에 대한 깊은 이해를 바탕으로
정확하고 실용적인 분석을 제공합니다.

반드시 JSON 형식으로만 응답하세요.
```

## 토픽 추출 프롬프트

```
아래는 {date} 호주 한인 카카오톡 그룹채팅의 대화 샘플입니다.
참여자 메타 정보: {meta_summary}

대화 샘플:
{conversation_sample}

다음 JSON 형식으로 오늘의 핵심 주제 3개를 추출하세요.
각 요약은 100자 이상이어야 합니다.

{{
  "topics": [
    {{
      "rank": 1,
      "title": "주제 제목 (10자 이내)",
      "summary": "3줄 요약 (각 줄 개행 구분, 총 100자 이상)"
    }}
  ]
}}
```

## 질문 분류 및 답변 프롬프트

```
아래는 호주 한인 커뮤니티에서 수집된 질문 군집입니다.
참여자 메타 컨텍스트: {meta_context}

질문 군집:
{question_clusters}

각 질문을 아래 7개 도메인 중 하나로 분류하고, 맥락 기반 답변을 생성하세요.

도메인 목록:
1. 통신·요금 — 통신사, 모바일 플랜, 인터넷 요금
2. 비자·이민 — 비자 종류, 영주권, 시민권, 이민법
3. 부동산·세금 — 렌트, 집 구매, 세금, 회계
4. 생활정보 — 쇼핑, 음식, 의료, 교통, 교육
5. 시사·뉴스반응 — 호주·한국 시사, 뉴스 반응
6. 잡담 — 일상 대화, 유머, 감정 표현
7. 기타 — 위 범주에 해당하지 않는 것

JSON 형식:
{{
  "question_analysis": [
    {{
      "domain": "도메인명",
      "representative_question": "대표 질문",
      "cluster_size": 숫자,
      "answer": "맥락 기반 답변 (50자 이상, 호주 실정 반영)",
      "confidence": "high|medium|low",
      "user_context": "메타 컨텍스트 요약 (선택)"
    }}
  ]
}}
```

## 자기검증 프롬프트

```
아래 분석 결과를 검증하세요.

분석 결과:
{analysis_result}

다음 기준으로 검증하고 JSON으로 응답하세요:
1. 주제 요약이 각 100자 이상인가?
2. 7개 도메인 분류가 유효한가?
3. 답변이 호주 실정을 반영하는가?
4. 답변 신뢰도가 medium 이상인가?

{{
  "validation": {{
    "topics_valid": true|false,
    "domains_valid": true|false,
    "answers_contextual": true|false,
    "overall": "pass|fail",
    "issues": ["문제점 목록 (없으면 빈 배열)"]
  }}
}}
```
